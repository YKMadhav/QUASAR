# Contributing to QUASAR

Thank you for your interest in contributing to **QUASAR (Quantum Understanding and Assessment System for Reliable Analysis)**.

## How to Contribute

Contributions are welcome in the following areas:

- Quantum circuit analysis
- Feature engineering
- Noise simulation
- Machine learning improvements
- Explainable AI (SHAP & LIME)
- Performance optimization
- Documentation
- Bug fixes

## Development Guidelines

- Follow existing project structure.
- Write clear, documented Python code.
- Keep functions modular.
- Ensure compatibility with the current dependencies listed in `requirements.txt`.
- Test changes before submitting.

## Pull Requests

Please include:

- Clear description of the change
- Reason for the change
- Testing performed
- Relevant screenshots (if UI changes)

## Code Style

- Follow PEP 8
- Use meaningful variable names
- Add docstrings where appropriate

Thank you for helping improve QUASAR.
